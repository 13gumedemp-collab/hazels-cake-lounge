# Hazel's Cake Lounge

Editorial, cinematic single-page website for Hazel's Cake Lounge — bespoke cakes & patisserie.

Static site, no build step. Three files:
- `index.html` — markup
- `styles.css` — styling (Fraunces + Jost, warm paper palette)
- `main.js` — scroll reveals, parallax, custom cursor, enquiry form

## Run locally
Open `index.html` in a browser, or serve it:

```bash
npx serve .
```

## Deploy
Hosted on Vercel (static). Pushes to `main` trigger automatic deploys.
